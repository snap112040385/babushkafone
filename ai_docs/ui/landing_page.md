# Лендинг - Спецификация UI/UX

## Обзор

Лендинг для Babushkafone (Бабушкафон) - VoIP-сервиса для русскоязычной диаспоры за рубежом. Страница полностью на русском языке и предназначена для конверсии посетителей в зарегистрированных пользователей.

## Дизайн-система

### Типографика
- **Семейство шрифтов**: Inter (Google Fonts)
- **Начертания**: 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold)
- **Базовые стили**: `font-sans antialiased`

### Цветовая палитра

#### Основные цвета
- **Индиго**: `indigo-600` - `indigo-900` - Основной цвет бренда для заголовков и кнопок
- **Фиолетовый**: `purple-600` - `purple-900` - Акцентный цвет для градиентов
- **Янтарный/Оранжевый**: `amber-300` - `orange-500` - Цвет для CTA-кнопок

#### Нейтральные цвета
- **Серая шкала**: `gray-50` - `gray-900` для текста и фонов
- **Белый**: Основной фон, текст кнопок

#### Акцентные цвета (секция боли)
- **Красный/Оранжевый**: `red-50`, `red-100`, `red-600` - Проблемы с банками
- **Оранжевый/Янтарный**: `orange-50`, `amber-50` - Проблемы с 2FA и стоимостью
- **Синий/Индиго**: `blue-50`, `indigo-50` - Госуслуги
- **Фиолетовый/Розовый**: `purple-50`, `pink-50` - Недвижимость и семья

### Паттерны компонентов

#### Кнопки
- **Основной CTA**: `bg-gradient-to-r from-amber-400 to-orange-500` с `rounded-full`, тень
- **Вторичный CTA**: `bg-white/10 backdrop-blur` с `rounded-full`
- **CTA тарифов**: `bg-gradient-to-r from-indigo-600 to-purple-600` с `rounded-xl`

#### Карточки
- **Карточки функций**: `bg-white rounded-2xl p-6 shadow-sm hover:shadow-lg`
- **Карточки боли**: Градиентные фоны с цветными границами, `rounded-2xl p-8`
- **Карточки тарифов**:
  - Стандартные: `bg-white/10 backdrop-blur rounded-3xl border border-white/20`
  - Выделенные: `bg-white rounded-3xl shadow-2xl` с бейджем "Популярный"

#### Секции
- Стандартные отступы: `py-20 px-6 lg:px-8`
- Максимальная ширина контейнера: `mx-auto max-w-7xl`

## Секции страницы

### 1. Hero-секция
- **Фон**: Градиент `from-indigo-900 via-purple-900 to-indigo-800`
- **Паттерн-оверлей**: SVG точечный паттерн с 40% прозрачностью
- **Компоненты**:
  - Навигационная панель с логотипом, ссылками, кнопками авторизации
  - Главный заголовок с градиентным выделением текста
  - Подзаголовок с ценностным предложением
  - Два CTA: Основной (Попробовать бесплатно) и Вторичный (Как это работает)
  - Индикаторы доверия (5 мин бесплатно, без подписки, крипта/карты)
  - Мокап телефона с активным звонком

### 2. Секция болей
- **Назначение**: Обращение к болевым точкам клиентов и создание срочности
- **Раскладка**: 3-колоночная сетка на десктопе, адаптивная
- **Карточки**: 6 карточек болевых точек с иконками и описаниями
- **Описанные боли**:
  1. Банки не отвечают в мессенджерах
  2. 2FA требует российский номер
  3. Роуминг дорогой
  4. Госорганы принимают только звонки
  5. Управление недвижимостью требует звонков
  6. Пожилые родственники не пользуются интернетом

### 3. Секция функций
- **Назначение**: Презентация функций решения
- **Раскладка**: 4-колоночная сетка функций + 3-колоночный ряд преимуществ
- **Ключевые функции**:
  - Звонки из браузера
  - Российский виртуальный номер
  - Низкие тарифы ($0.04/мин)
  - Оплата криптой и картами
- **Дополнительные функции**:
  - Кристально чистый звук
  - Запись звонков
  - Интерфейс на русском языке

### 4. Секция "Как это работает"
- **Назначение**: Показать простой процесс онбординга
- **Раскладка**: 3-шаговый горизонтальный поток с соединительной линией
- **Шаги**:
  1. Регистрация (только email)
  2. Пополнение баланса (карта, крипта, Wise)
  3. Начало звонков

### 5. Секция тарифов
- **Фон**: Темный градиент (`gray-900 via-indigo-900 to-purple-900`)
- **Раскладка**: 3-колоночные карточки тарифов
- **Тарифы**:
  1. **Поминутная оплата**: $0.04/мин на стационарные, $0.06/мин на мобильные
  2. **Стартовый** (Выделенный): $20/месяц - Российский номер + 200-300 мин
  3. **Бизнес**: $35/месяц - Российский номер + 450-675 мин
- **Способы оплаты**: Visa, Mastercard, USDT, BTC, Wise, Revolut

### 6. Секция FAQ
- **Назначение**: Ответы на частые вопросы
- **Компонент**: Раскрывающийся аккордеон `<details>`
- **Охваченные вопросы**:
  - Как работает виртуальный российский номер
  - Легальность
  - Заблокированные российские карты
  - Качество звонков
  - Необходимость VPN
  - Поддержка звонков на стационарные

### 7. Секция CTA
- **Назначение**: Финальный толчок к конверсии
- **Фон**: Градиент от индиго к фиолетовому
- **Компоненты**: Заголовок, подзаголовок, два CTA (Начать бесплатно, Telegram)

### 8. Футер
- **Фон**: `gray-900`
- **Раскладка**: 4-колоночная сетка
- **Секции**: Бренд, Продукт, Поддержка, Правовая информация
- **Соцсети**: Иконка Telegram

## Адаптивное поведение

### Брейкпоинты
- **Мобильный**: По умолчанию (< 768px)
- **Планшет**: `md:` (768px+)
- **Десктоп**: `lg:` (1024px+)

### Ключевые адаптивные изменения
- Навигация: Ссылки скрыты на мобильном, показаны на `md:`
- Hero: Одна колонка на мобильном, 2-колоночная сетка `lg:`
- Мокап телефона: Скрыт на мобильном, показан `lg:block`
- Секция болей: Одна колонка на мобильном, 2-3 колонки `md:`/`lg:`
- Функции: 1 колонка на мобильном, 2 колонки `md:`, 4 колонки `lg:`
- Тарифы: Вертикально на мобильном, 3 колонки `md:`

## Доступность

- Семантическая структура HTML с правильными заголовками
- SVG-иконки с корректным viewBox
- Соблюдение контрастности текст/фон
- Интерактивные элементы с hover-состояниями
- Раскрывающийся FAQ на нативном элементе `<details>`

## Анимации

- Пульсирующая анимация на индикаторе статуса: `animate-pulse`
- Hover-переходы на кнопках и карточках: `transition`
- Поворот шеврона в FAQ: `group-open:rotate-180 transition`
