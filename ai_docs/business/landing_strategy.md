# Лендинг - Бизнес-стратегия

## Обзор

Лендинг Babushkafone является основным инструментом конверсии для привлечения новых пользователей из сообщества русскоязычной диаспоры за рубежом.

## Целевая аудитория

На основе GTM-манифеста, лендинг нацелен на:

### Основной ICP (Идеальный профиль клиента)
- **Демография**: Русскоязычные профессионалы 25-45 лет
- **Локация**: Турция, Казахстан, Грузия, Армения, Сербия, Израиль, Германия, ОАЭ, ЕС, Северная Америка, Латинская Америка
- **Доход**: $2,000-6,000/месяц
- **Занятость**: IT-специалисты, предприниматели, удаленные работники, владельцы бизнеса

### Ключевые характеристики
- 83% в возрасте 18-44 лет
- Высокая цифровая грамотность
- Сохраняют российское гражданство и активы
- 45% испытывают банковские проблемы за рубежом
- 14% - маятниковые мигранты, регулярно посещающие Россию

## Ценностное предложение

### Главный заголовок
"Звоните в Россию из любой точки мира"

### Поддерживающие сообщения
- Звонки из браузера (без установки приложения)
- Российские номера для 2FA
- В 10 раз дешевле роуминга ($0.04/мин)

### Индикаторы доверия
- Работает в 50+ странах
- 5 бесплатных минут для пробы
- Без обязательной подписки
- Оплата криптой и картами

Лендинг явно обращается к 6 ключевым болевым точкам:

| Боль | Сообщение | Эмоциональный триггер |
|------|-----------|----------------------|
| Банки требуют телефонные звонки | Сбербанк, Тинькофф, ВТБ требуют звонков для решения проблем с картами | Фрустрация, срочность |
| 2FA требует российский номер | Госуслуги, банки требуют SMS-коды | Страх потери доступа |

| Пожилые родственники офлайн | Родители/бабушки без смартфонов | Чувство вины, семейный долг |

## Стратегия конверсии

### Основные CTA
1. **"Попробовать бесплатно"** - Точка входа с низким барьером
2. **"Начать бесплатно"** - CTA регистрации
3. **"Выбрать"** - Выбор тарифного плана

### Воронка конверсии

```
Осведомленность -> Интерес -> Рассмотрение -> Конверсия
       |              |             |             |
  Hero-секция -> Секция болей -> Функции -> Тарифы
                + Как это работает -> FAQ -> CTA-секция
```

### Предложение бесплатного пробного периода
- 5 бесплатных минут звонков
- Без привязки карты
- Регистрация только по email

## Ценовая стратегия (отображается на лендинге)

### Поминутная оплата
- $0.04/мин на стационарные
- $0.06/мин на мобильные
- Без месячной платы
- Кредиты не сгорают
- Минимальное пополнение $5

### Стартовый пакет (Выделенный)
- $20/месяц
- Российский номер +7 включен
- 200-300 минут
- SMS для 2FA
- Отмена в любое время

### Бизнес-пакет
- $35/месяц
- Российский номер +7 включен
- 450-675 минут
- Запись звонков
- Приоритетная поддержка

### Способы оплаты
Акцент на доступности для диаспоры с заблокированными российскими картами:
- Visa/Mastercard (международные)
- USDT, BTC (криптовалюта)
- Wise, Revolut (цифровые кошельки)

## Конкурентное позиционирование

### Против традиционного роуминга
- "В 10 раз дешевле роуминга"
- Прямое сравнение: $0.50-2/мин vs $0.04/мин

### Против мессенджеров (WhatsApp, Telegram)
- "Нельзя позвонить на российские стационарные, в бизнес, госорганы"
- Позиционирование как дополнение, а не замена

### Против других VoIP-сервисов
- Интерфейс на русском языке
- Оплата криптой
- Российские виртуальные номера для 2FA
